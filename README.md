
# Flight Simulator 2020 - icon a5 Rotax 915 iS

This mod is experimental. It has been made by gamers, not experts !

## Downloads

* Download the [nightly build](https://a5-915.r9r.dev/nightly.zip) made everyday at 2a.m. UTC
* Download the [latest build](https://a5-915.r9r.dev/latest.zip) made after every push

![nightly](https://github.com/r9r-dev/fs2020-icona5-rotax915/workflows/nightly/badge.svg) ![latest](https://github.com/r9r-dev/fs2020-icona5-rotax915/workflows/latest/badge.svg)

The release is the only one tested and verified before publication. The nightly and latest versions may sometimes not work but allow you to test our latest modifications. The nightly and latest versions are hosted in a datacenter in Paris and may be slower or faster to download depending on your region.
